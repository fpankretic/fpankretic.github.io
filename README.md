# Portfolio Website

This project is a personal portfolio website with a bold masculine-pink visual theme and a single-page layout.  
It includes dedicated sections for About Me, Work Experience, and external links, with editable content stored directly in the homepage file for quick updates.

The page is built with SEO in mind and includes metadata for search and social previews (title, description, canonical URL, Open Graph, and Twitter cards).  
Company logos can be added per experience entry, and profile/work content can be customized without changing the page structure.

## Local Development

Run from the project root:

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
```
